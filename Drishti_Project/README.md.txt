👁️ Dristi – AI Vision Assistant

📌 Description

Dristi is an AI-based project that detects objects using a camera and provides voice output to help users understand their surroundings.

🧠 Technologies Used

- Python
- OpenCV
- TensorFlow / Keras
- pyttsx3

🚀 Features

- Object detection
- Voice output
- Real-time processing

▶️ How to Run

1. Install libraries
2. Run:
   python main.py

📁 Files

- main.py → main program
- labels.txt → object labels
- model → trained model
