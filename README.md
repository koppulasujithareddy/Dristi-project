Drishti.

Overview

Drishti is an AI-powered assistive system designed to help visually impaired individuals identify Indian currency notes and receive audio feedback. The project aims to improve accessibility and independence through computer vision and voice assistance technologies.

Features

- Indian currency note recognition
- Voice output in English and Telugu
- User-friendly interface
- Fast and accurate currency identification
- Accessibility-focused solution

Technologies Used

- Python
- Machine Learning
- OpenCV
- NumPy
- Text-to-Speech (TTS)
- VS code

Project Description

Drishti was developed to assist visually impaired individuals in identifying Indian currency notes independently. Using image processing and computer vision techniques, the system detects the denomination of currency notes and provides audio feedback to the user in multiple languages.

Installation

1. Clone the repository:
   git clone <repository-link>

2. Install required packages:
   pip install -r requirements.txt

3. Run the application:
   python main.py

Future Enhancements

Impact

Drishti helps visually impaired users perform daily financial transactions more confidently and independently through AI-based assistance.
